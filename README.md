# 🛒 E-Commerce Web Application

A full-stack E-Commerce Web Application built using Flask, Python, HTML, CSS, and SQLite. The application allows users to browse products, add them to the cart, place orders, and provides an admin panel to manage products.

## 🚀 Live Demo

https://ecommerce-flask-app-yj0r.onrender.com

## 📌 Features

- 👤 User Registration and Login
- 🔐 Secure User Authentication
- 🛍️ Browse Products
- 🛒 Add Products to Cart
- 💳 Checkout Functionality
- 👨‍💼 Admin Dashboard
- ➕ Add New Products
- ✏️ Edit Existing Products
- ❌ Delete Products
- 📦 Order Management
- 💾 Database Integration using SQLite

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- SQLite
- Jinja2

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/saisnehitha28/ecommerce-flask-app.git
```

2. Navigate to the project folder

```bash
cd ecommerce-flask-app
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
